Front-End Framework

React

What is React?

  • A JavaScript library for building user interfaces
  • Developed and maintained by Meta (Facebook)
  • Uses a component-based architecture
  • Virtual DOM for efficient rendering
  • One of the most popular front-end frameworks

Front-End Framework

Why We Chose React

  • Component Reusability — Build once, use everywhere across the application
  • Large Ecosystem — Extensive libraries and community support (UIverse, Material UI)
  • Performance — Virtual DOM ensures smooth, fast user experiences
  • Industry Standard — Widely used in professional development; great skill to learn
  • Easy Integration — Works seamlessly with UI libraries like Uiverse and Material UI

UI Libraries

UIverse

Open-source UI elements library with beautiful, ready-to-use components.

Material UI / React Icons

Google's Material Design components and a comprehensive icon library.

Why These Libraries?

These libraries are "goods sa React" — they integrate seamlessly and speed up development.

Inspirations & Aesthetics

Magical Origins

The globally beloved Harry Potter universe serves as our primary inspiration. We blended the mystical, deep aesthetics of the wizarding world—specifically the four iconic Hogwarts houses—with modern web design principles.

Interactive UI Layouts

We leverage interactive hover-to-reveal cards and dynamic expansion to create an immersive experience. This 'magical reveal' mechanic allows users to discover content seamlessly—keeping the layout elegant and uncluttered while providing deep interactive engagement at a single glance.
